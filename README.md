# 🎵 Spotify Wrapped 2025 Analysis

## 📊 Overview
This project analyzes Spotify Wrapped 2025 top songs using R, SQL, and Tableau. The analysis explores streaming performance, genre trends, and audio features such as energy and danceability.

## 🛠️ Tools Used
- R (tidyverse, ggplot2)
- SQL (SQLite)
- Tableau Public
- Google Sheets

## 📁 Dataset
- Spotify Wrapped 2025 Top Songs Dataset
- Updated March 2026

## 📈 Key Insights
- Pop music and its subgenres dominate streaming performance.
- Streaming success is concentrated among a small group of artists.
- High-energy songs do not consitently outperform lower-energy songs.
- Explicit songs shoe a slight advantage in average streams.

## 💾 SQL Analysis
The SQL portion focused on:
- Top streamed songs
- Genre distribution
- Audio fature filtering
- Explicit vs non-explicit comparison
